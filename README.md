# docker-python-template
A template to quickly build a python development environment.

## Environment
- Docker version 19.03.12, build 48a66213fe
- docker-compose version 1.26.2, build eefe0d31

## Stack
Python 3.8.6

## Run python  
```
$ docker-compose exec python bash
```

```
# python --version
Python 3.8.6

# python sample.py
Hello, world!
```
